# React Native Crash Course

A look at what React Native is, CLI setup, initialize application and some key aspects of the React Native framework.

## Segments:
* Components, State & Props
* View & Text
* StyleSheet
* Touchable Elements
* ListView
* Navigator
* Button
* Events

### UI Component Examples
* TextInput, Picker, Switch, Slider
* Touchable Elements
* ListView & ScrollView
* Alerts, Modals, ProgressBars
* StatusBar
* TabBarIOS
* Toolbar Android

### APIs for Device Interaction
* CameraRoll
* AsyncStorage
* Geolocation
* ImageEditor
* PushNotifications
* Vibration
* Share/Messages

### Dev Environment & Specs
* Windows
  - Android Studio
  - Android SDK (7.0 Nougat)
  - Android AVD

* Mac
  - Xcode
  - Simulator
